# Vibe Coder

Voice-first coding assistant powered by Deepgram.

## Features

- Start dictation with `Cmd+Shift+D` (Mac) or `Ctrl+Shift+D` (Windows/Linux)
- Start voice agent with `Cmd+Shift+A` (Mac) or `Ctrl+Shift+A` (Windows/Linux)

## Requirements

- Deepgram API key
- Microphone access
